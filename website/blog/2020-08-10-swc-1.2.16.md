## Bugfixes

### typescript class properties ([3951](https://github.com/swc-project/swc/pull/951))

This is a bugfix for a new feature introduced in [swc][]@`v1.2.15`.

An example of affected code is

```ts
class B {}

class A extends B {
  foo = new Subject();
  b = this.a;

  constructor(readonly a) {
    super();
    this.foo.subscribe();
  }
}
```
